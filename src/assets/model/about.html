MailHog Desktop


Versão: 1.1.0

O MailHog-Desktop é um frontend desenvolvido em Electron para o projeto MailHog.

Atualmente o projeto Mailhog possui duas aplicações, uma para gerenciar um cliente SMTP falso para receber e-mails chamada mailhog e outra para realizar envio de e-mails chamada mhsendmail.

Esta ferramenta foi desenvolvida com a tecnologia Electron e está registrada sob a licença MIT.
